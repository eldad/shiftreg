# Shift Register Control

This is a tiny flipperzero app to manually control a 74HC595 shift register.

Connections:

| 74HC595      | FlipperZero GPIO Pin |
|--------------|----------------------|
| 14 SER       | 2 A7                 |
| 13 OE        | 3 A6                 |
| 12 RCLK      | 4 A4                 |
| 11 SRCLK     | 5 B3                 |
| 10 SRCLR     | 6 B2                 |
